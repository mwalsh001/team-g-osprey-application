# TeamG-Osprey-Application
Our Osprey School Benchmarking System is a web application that allows school users to enter, edit, and analyze annual benchmarking school data. School users can submit data related to their school's enrollment through multiple, organized forms, and administrators can enter data for any school. Users are also able to see on their dashboard relevant charts displaying their school's enrollment data and compare their school's data to data from different regions. Some information tracked includes: enrollment trends, retention rates, attrition rates, and inquiry trends. Administrators can select any school and view the dashboard for that school. They are also given the ability to create new school users.
